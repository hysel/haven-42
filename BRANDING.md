# Haven 42 Brand Standard

## Product Identity

- Product name: Haven 42
- Tagline: Your private, local AI station.
- Canonical repository: `hysel/haven-42`
- Release artifact prefix: `haven-42-<version>`

Haven 42 describes the complete local-first product direction: software-engineering workflows, repository-optional general AI tasks, local provider capabilities, typed artifacts, evidence, and the planned local web experience.

## Product Identity Policy

All product-specific identifiers use the Haven 42 name directly. This includes:

- Guided-menu workflow IDs and script filenames.
- Shared-assets and managed MLX environment directories.
- Release artifact names and temporary session paths.
- Provider client identifiers, canonical links, and public documentation.

Do not add alternate product identities, compatibility aliases, or deprecated branding to scripts, documentation, package names, configuration, examples, or generated artifacts.

## Repository

The canonical repository and wiki are `hysel/haven-42` and `hysel/haven-42.wiki`. Local checkouts, remotes, documentation links, and release automation should use those canonical identities.
